## Installation

CRAN version

```{r}
install.packages("mlr3tuning")
```

Development version

```{r}
remotes::install_github("mlr-org/mlr3tuning")
```

Further documentation can be found in the [mlr3book](https://mlr3book.mlr-org.com/tuning.html).
